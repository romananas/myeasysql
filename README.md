
# My Easy Sql

MyEasySQL is a student projet of [Go](https://go.dev/) package intended to simplify [SQL](https://sql.sh/) querys and executions




## Installation

Install myeasysql with golang

```bash
  go get github.com/romananas/myeasysql@latest
```
